使用流程

简单的三步使用方式 

sequenceDiagram
    participant U as 用户
    participant OC as OpenClaw
    participant S as Skill
    participant W as Workspace
    
    U->>OC: 自然语言请求
    OC->>S: 匹配对应 Skill
    S->>W: 读取工作区文件
    S->>S: 执行分析/处理
    S-->>OC: 返回结果
    OC-->>U: 输出报告/建议

典型使用场景

  1. 新代码库熟悉
    用 code-analyzer 快速理解项目架构、数据流、业务规则
  2. Code Review 准备
    用 java-code-review 自动检查规范、异味、安全问题
  3. 文档质量审计
    用 docs-improver 评估完整性、准确性，生成缺失文档
  4. 技术方案生成
    用 solution-architect 从 PRD 生成完整技术方案
  5. 演示文稿制作
    用 ppt-generator 快速生成炫酷的技术分享 PPT
