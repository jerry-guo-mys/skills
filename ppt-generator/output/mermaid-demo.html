时序图：OAuth 2.0 授权

清晰描绘系统间的交互与时序关系

sequenceDiagram
    autonumber
    participant User as 用户
    participant App as 第三方应用
    participant Auth as 授权服务器
    participant Res as 资源服务器

    User->>App: 点击微信登录
    App->>Auth: 重定向至授权页面(携带 client_id)
    Auth-->>User: 展示授权同意书
    User->>Auth: 用户点击同意
    Auth-->>App: 返回授权码
    App->>Auth: 携带 Code 和 Secret 请求 Token
    Auth-->>App: 下发 Access Token
    App->>Res: 携带 Token 请求用户信息
    Res-->>App: 返回头像与昵称
    App-->>User: 登录成功并展示首页
