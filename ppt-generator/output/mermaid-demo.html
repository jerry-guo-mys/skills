流程图示例

系统架构流程图 

flowchart TD
    A[用户请求] --> B{认证？}
    B -->|是 | C[API 网关]
    B -->|否 | D[返回 401]
    C --> E[微服务 A]
    C --> F[微服务 B]
    E --> G[数据库]
    F --> G
    G --> H[返回结果]

时序图示例

用户登录流程 

sequenceDiagram
    participant U as 用户
    participant F as 前端
    participant A as API
    participant D as 数据库
    
    U->>F: 输入账号密码
    F->>A: POST /login
    A->>D: 查询用户
    D-->>A: 返回用户信息
    A-->>F: 返回 Token
    F-->>U: 登录成功

类图示例

领域模型类图 

classDiagram
    class User {
        +String id
        +String name
        +String email
        +login()
        +logout()
    }
    class Order {
        +String id
        +Date createdAt
        +List~Item~ items
        +checkout()
    }
    class Item {
        +String productId
        +int quantity
        +decimal price
    }
    User --> Order : places
    Order --> Item : contains

状态图示例

订单状态流转 

stateDiagram-v2
    [*] --> Pending : 创建订单
    Pending --> Paid : 支付成功
    Pending --> Cancelled : 取消订单
    Paid --> Shipping : 发货
    Shipping --> Delivered : 签收
    Delivered --> [*]
    Paid --> Refunded : 申请退款
    Refunded --> [*]

甘特图示例

项目进度计划 

gantt
    title 项目开发计划
    dateFormat  YYYY-MM-DD
    section 需求
    需求分析 :a1, 2026-03-01, 7d
    原型设计 :after a1, 5d
    section 开发
    前端开发 :2026-03-10, 14d
    后端开发 :2026-03-10, 14d
    section 测试
    单元测试 :2026-03-20, 7d
    集成测试 :2026-03-25, 5d
